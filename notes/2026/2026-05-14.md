### [Suggestion: Update Node.js version support in code-explorer](https://github.com/eslint/code-explorer/issues/378)

**TSC Summary:** Node.js 20 reached End-of-Life on April 30, 2026, but web apps such as `code-explorer` and `eslint.org` still depend on it. The TSC was asked which Node.js version to upgrade to and what the recommended first step would be, given that some setup changes may be needed on Netlify.

**Discussion Points:**
- @mdjermanovic favored switching directly to Node.js 24.
- The team noted that some sites have local configs, so the exact transition steps will need to be worked out per-app.

**Resolution:** The TSC agreed to upgrade the Netlify apps to Node.js 24. Technical details will be determined on a per-app basis.

### [fix: replace Intl API usage for broader Node.js compatibility](https://github.com/eslint/eslint/pull/20396)

**TSC Summary:** The PR aimed to remove ESLint's dependency on the `Intl` global, but the change turned out to be trickier than expected.

**Discussion Points:**
- @nzakas was in favor of not moving forward with the PR.
- @mdjermanovic and @fasttime agreed the team should keep using `Intl` and instead update the documented prerequisites.

**Resolution:** The TSC agreed not to move forward with the PR and to update the documented prerequisites for running ESLint instead. @nzakas will close the PR.
